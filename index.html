Raio-X Financeiro

Solução de Automação para Gestão Financeira no Google Sheets™

Sobre o Aplicativo

O Raio-X Financeiro é um script de automação (Add-on/Script) desenvolvido para operar exclusivamente dentro do ambiente do Google Sheets. Seu propósito principal é facilitar o controle financeiro pessoal através de formulários personalizados que padronizam a entrada de dados.

Diferente de softwares financeiros externos, o Raio-X Financeiro não armazena seus dados em servidores de terceiros. Ele atua como uma camada de interface entre o usuário e a própria planilha do usuário, garantindo privacidade total.

Principais Funcionalidades:

  • Registro automatizado de despesas, receitas e investimentos.
  • Geração de relatórios e dashboards visuais dentro da planilha.
  • Interfaces de formulário (pop-ups) para evitar erros de digitação manual nas células.

Uso de Dados e Permissões (Escopos)

Para fornecer as funcionalidades descritas, o aplicativo solicita as seguintes permissões específicas do Google, com as respectivas justificativas:

1. Gerenciamento de Planilhas

https://www.googleapis.com/auth/spreadsheets

Por que precisamos disso: O script precisa dessa permissão para escrever os dados financeiros que você insere nos formulários diretamente nas células das abas "Despesas" e "Receitas". Também utilizamos para ler configurações salvas na aba "Configurações" para personalizar as categorias disponíveis.

2. Interface de Usuário Externa

https://www.googleapis.com/auth/script.container.ui

Por que precisamos disso: Esta permissão é estritamente necessária para exibir as janelas modais (pop-ups) e a barra lateral personalizada onde os formulários de lançamento são apresentados. Sem isso, o usuário teria que digitar os dados manualmente nas células, perdendo a função de automação.

Compromisso com a Privacidade

O Raio-X Financeiro adere à Política de Dados do Usuário dos Serviços de API do Google, incluindo os requisitos de uso limitado.

  • Sem Transferência de Dados: O aplicativo não transfere dados da sua planilha para nenhum servidor externo.
  • Processamento Local: Todo o processamento de cálculos ocorre dentro da infraestrutura da sua conta Google.
  • Acesso Restrito: O desenvolvedor não tem acesso aos dados inseridos nas suas planilhas.
