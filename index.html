Raio-X Financeiro

Ferramenta de Automação e Controle Financeiro para Google Sheets™

Funcionalidade do Aplicativo

O Raio-X Financeiro não é uma plataforma externa, mas sim um Add-on (complemento) de produtividade que roda integrado diretamente à planilha do Google Sheets do usuário. Ele transforma uma planilha em branco em um sistema robusto de gestão financeira através das seguintes funções:

✅ Interface de Entrada de Dados (Data Entry)

O aplicativo substitui a digitação manual nas células por formulários HTML amigáveis (pop-ups). Isso padroniza a entrada de dados, prevenindo erros de formatação em datas e valores monetários.

✅ Automação de Lançamentos

Ao enviar um formulário, o script processa as informações e as distribui automaticamente para as abas corretas ("Despesas", "Receitas", "Investimentos"), mantendo o banco de dados do usuário sempre organizado sem intervenção manual.

✅ Dashboards Dinâmicos

O sistema processa os dados brutos inseridos pelo usuário para gerar visualizações gráficas e relatórios de fluxo de caixa em tempo real dentro da própria planilha.

Uso de Dados e Justificativa de Permissões

Para executar as funções acima descritas, o aplicativo requer acesso estritamente necessário aos seguintes escopos do Google Workspace. Abaixo detalhamos como cada permissão é utilizada:

1. Ver, editar, criar e excluir arquivos do Google Planilhas

https://www.googleapis.com/auth/spreadsheets

Finalidade Específica:

  • Leitura (Read): O aplicativo lê as configurações definidas pelo usuário na aba "Configurações" (como categorias personalizadas de gastos) para preencher as listas suspensas nos formulários.
  • Escrita (Write): Quando o usuário clica em "Salvar" no formulário, o aplicativo precisa desta permissão para escrever os novos dados na próxima linha vazia das abas de registro ("Despesas" ou "Receitas").
  • Edição: Permite ordenar os lançamentos por data e atualizar valores caso o usuário edite um registro antigo através da interface do script.

Justificativa: Sem este escopo, o aplicativo não consegue interagir com a planilha, tornando impossível a automação de registros.

2. Exibir e executar conteúdo da Web de terceiros em prompts e barras laterais

https://www.googleapis.com/auth/script.container.ui

Finalidade Específica:

  • Interface Gráfica: Utilizamos este escopo para renderizar os formulários HTML, CSS e JavaScript personalizados que aparecem como janelas modais (pop-ups) ou barras laterais dentro da planilha.
  • Interatividade: Permite a criação de botões, menus de seleção e alertas de confirmação (ex: "Lançamento salvo com sucesso") que melhoram a experiência do usuário.

Justificativa: Este escopo é essencial para fornecer uma interface visual amigável, evitando que o usuário precise lidar diretamente com códigos ou fórmulas complexas.

Política de Dados e Privacidade

O Raio-X Financeiro compromete-se com a transparência e a segurança dos seus dados:

  • Armazenamento Local: O aplicativo não possui banco de dados externo. Todos os seus dados financeiros permanecem salvos exclusivamente dentro da sua própria planilha do Google Sheets.
  • Sem Compartilhamento: Não compartilhamos, vendemos ou transferimos suas informações para terceiros. O desenvolvedor não tem acesso ao conteúdo das suas planilhas.
  • Execução Segura: Todo o código é executado dentro do ambiente seguro do Google Apps Script (Google Cloud).
