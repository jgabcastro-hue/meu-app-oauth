Política de Privacidade

1. Informações Gerais

Este aplicativo (script de automação para Google Sheets) foi desenvolvido para facilitar o controle financeiro pessoal. Ele processa dados diretamente na planilha do usuário.

2. Coleta e Uso de Dados

Dados Processados: O script acessa apenas as informações fornecidas pelo usuário através dos formulários internos (Gastos, Receitas, Investimentos).

Finalidade: Os dados são utilizados exclusivamente para organizar registros financeiros dentro das abas da própria planilha do usuário.

Armazenamento: Nossos scripts não armazenam e não transmitem seus dados para nenhum servidor externo. Todas as informações permanecem dentro da conta Google Drive do usuário.

3. Permissões Requeridas (Escopos)

Para funcionar corretamente, o script solicita as seguintes permissões:

  • Ver e gerenciar planilhas: Necessário para inserir os lançamentos nas abas "Despesas", "Receitas" e "Investimentos".
  • Exibir interface de usuário: Necessário para abrir as janelas de formulário em HTML.

4. Compartilhamento de Dados

Não compartilhamos, vendemos ou alugamos quaisquer dados dos usuários para terceiros. O acesso aos dados é restrito exclusivamente ao proprietário da planilha.

5. Segurança

O script utiliza a infraestrutura de segurança do próprio Google (Google Apps Script) para garantir a proteção das informações.
